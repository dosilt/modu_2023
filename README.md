## 환경 셋팅


```
conda create -n <env_name> python=3.9.18

```

## 라이브러리 설치
```
git clone <repository>
cd modu_2023

pip install -r requirement.txt

git clone https://github.com/huggingface/peft.git
cd peft
python setup.py install
cd ..
```
